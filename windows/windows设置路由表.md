外网：

```
网关：192.168.0.1
本机ip：192.168.0.2
```

内网：

```
网关：172.16.0.1
本机ip：172.16.0.2
```

 

1、在终端里输入 

```
route print -4 #查看IPv4路由表
```

2、删掉所有的默认路由：

```
route delete 0.0.0.0
```

4、再重新添加需要的默认路由，一般是走公网的网关：

 

```
route add 0.0.0.0 mask 0.0.0.0 192.168.0.1
```

 

5、再加上内网的路由，到内网服务器地址，走内网网关：

```
route add 10.0.0.0 mask 255.0.0.0 172.16.0.1
```
